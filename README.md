I am a Ph.D. researcher at the **University of Genoa** in the **RICE Lab**, working on mixed-reality interfaces for heterogeneous multi-robot teams.

My research asks how human operators can supervise, coordinate, and intervene in robot teams through spatial interfaces instead of fragmented screens, camera feeds, and control panels. I work across mixed reality, human-robot interaction, teleoperation, shared tasking, field robotics, and robot intelligence for multi-robot systems.

## What I Work On

- **Mixed-reality robot supervision**: interfaces for monitoring robot state, maps, sensor data, and mission progress in one spatial workspace.
- **Multi-robot coordination**: shared maps, task allocation, goal-level control, and operator intervention for heterogeneous robot teams.
- **Teleoperation and remote operation**: moving between supervisory control and direct robot control when the environment or mission requires it.
- **VLA for mobile robot navigation**: exploring how vision-language-action models can support navigation, multi-robot navigation, and coordination in shared environments.
- **Robot intelligence through MR**: making scene understanding, learned robot skills, and robot intent legible to human partners.

## Current Focus

- **HORUS**: a mixed-reality platform for managing mobile robot teams through shared maps, robot state visualization, task tools, and teleoperation.
- **Saxion research-abroad period**: exploring vision-language-action models for mobile robot navigation, with a focus on multi-robot navigation and coordination.
- **Field robotics validation**: testing MR interfaces across platforms such as Spot, wheeled robots, firefighting drones, and humanoid robots.
